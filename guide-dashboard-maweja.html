🧭 Structure du Dashboard — La barre latérale
Le Dashboard est accessible à l'adresse maweja.net/admin. Une fois connecté, la barre latérale gauche contient tous les menus regroupés en 4 grandes sections.
  • ●
    Vue générale — Dashboard + Commandes
  • ●
    Gestion — Agents, Restaurants, Boutiques, Zones, Services
  • ●
    Clients — Clients, Chat, Notifications
  • ●
    Finance & Système — Finance, Marketing, Promo, Pub, Comptes, Paramètres

📌 Les 4 statistiques principales

    Commandes du jour — Nombre total de commandes reçues depuis minuit
  • 💵
    Revenu du jour — Somme totale générée en USD depuis minuit

📌 Fonctionnalités disponibles
  • ✓
    Ajouter, modifier ou supprimer un restaurant (nom, adresse, type de cuisine, photo)
  • %
    Définir le taux de commission — Appliqué automatiquement sur les prix affichés aux clients
  • 🍕
    Gérer le menu complet : articles, prix de base, photos, disponibilité
  • 🔄

  • ⏱
    Configurer les temps de préparation et de livraison estimés
💡
Commission : Les prix dans le menu admin sont les prix de base HT. La commission est automatiquement ajoutée sur le prix visible aux clients. Exemple : article à $10 avec 15% de commission = $11.50 pour le client.

6 — Catégories Restaurants
Organisation et filtres par type de cuisine
📌 Rôle de cette section
Les catégories permettent aux clients de filtrer les restaurants par type de cuisine. Elles apparaissent comme des onglets horizontaux en haut de la section Restaurants dans l'app.
  • ✓
    Créer une catégorie (nom + icône emoji + couleur)
  • ↕
    Réordonner par glisser-déposer (ordre = ordre d'affichage)
  • ✏️
    Modifier le nom ou l'icône
  • 🗑
    Supprimer une catégorie inutilisée
Exemples de catégories
🍲Congolaise 8 restaurants
🍔Fast-Food 5 restaurants
🍕Pizzeria 3 restaurants
